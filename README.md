# проверка email(Легкая логика)

Проект по Introduction to speciality 
Простой и быстрый код для проверки правильно ли написан email

# чем проект отличается от других
несколько видов проверки в одном коде:
-автоматический убирает лишние пробелы по краям
-проверяет есть ли @ в данном тексте и делит его на 2 части
-проверяет есть ли точка после @
-смотрит правильно ли стоят точки
-программа работает в цикле и дает возможность проверить несколько email-ов

# инструкция как установить и использовать
1.Скачайте Python версии 3+
2.Скачайте файл "a.py" или же скопируйте код и вставьте в пайтон
3.ну и самое главное запускайте код в терминале

# пример как должен писаться input
bahakrutoy@gmail.com -> это правильно
bahakrutoy@gmailcom -> это не правильно потому что нету точки
bahakrutoy@gmail..com -> это не правильно потому что тут 2 точки
@gmailcom -> ошибка поскольку тут нету начала
выход -> это выход из программы
